Istruzioni

1. Installazione
- Consulta il file  word rinominato "Diario_Ate" (Diario\Backend\Diario)  alla sezione "1.2 tecnologie utilizzate" per installare tutte le tecnologie utilizzate nel progetto.
- Crea i database in PostgreSQL e MongoDB seguendo la struttura riportata nella sezione "2.3 Classi modello".
- Installa le variabili di ambiente 

2. Configurazione
Apri il file application.properties e imposta i seguenti parametri:
- Per PostgreSQL: Imposta l'url, l'username e la password per la connessione a PostgreSQL.
- Per MongoDB: Imposta solo l'url per la connessione a MongoDB
